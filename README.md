# Martin Henderson — Fan Meet & Greet

Static multi-page site built with [Vite](https://vitejs.dev/).

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at `http://localhost:5173` (or the next available port).

## Build

```bash
npm run build
```

Output goes to `dist/`. Deploy the contents of `dist/` to any static host.

## Preview production build

```bash
npm run preview
```

## Font

Place `LayGrotesk_Medium.woff` in the `public/` folder so it is served at `/LayGrotesk_Medium.woff`. If the file is missing, the site will fall back to system fonts.

## Pages

- `/` — Home
- `/booking.html` — General booking
- `/meet-and-greet.html` — Meet & Greet booking
- `/fan-card.html` — Fan Card application
- `/report-scam.html` — Report scam / impersonation
